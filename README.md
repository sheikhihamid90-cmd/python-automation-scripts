# Python Automation Scripts

A collection of beginner-friendly Python automation utilities with a simple command-line interface.

## Included Scripts
- Word counter for text files
- Bulk file renamer
- CSV summary tool

## Getting Started
1. Make sure Python 3.10+ is installed.
2. Run the CLI:
   ```bash
   python main.py --help
   ```

## Examples
Count words in a file:
```bash
python main.py wordcount sample.txt
```

Rename files in a folder:
```bash
python main.py rename ./files invoice
```

Create a CSV summary:
```bash
python main.py csv-summary data.csv
```
